# ウィンドウをフレームレスにする

## メインプロセス

```javascript:main.js
const createWindow = () => {
  const mainWindow = new BrowserWindow({
    // フレームレス
    frame: false,
   
  });

  mainWindow.loadFile('index.html');
};
```

## ドラッグ可能にする

フレームレス・ウィンドウではタイトルバーが表示されなくなるので、レンダラープロセスでドラッグ可能な領域を用意しなければいけない場合もある。

```css:レンダラープロセス(css)
html,
body,
.container {
  height: 100%;
  overflow: hidden;
}

.container {
  display: flex;
  /* ドラッグ可能なゾーンとする */
  -webkit-app-region: drag;
}
```
